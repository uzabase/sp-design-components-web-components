# Checkbox
複数の選択肢から任意の項目を複数選択させたい時に使用します  
チェックボックスは単体で使用できます  
※1つを選択させたいときはラジオボタンを使用します

# Usage
- チェックボックスにはindeterminateがあります
    - indeterminateは全てのチェックボックスを一括で選択する際に使用します
    - 一括で選択した中で全てにチェックがついていない時の状態を指します
