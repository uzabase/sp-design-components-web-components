# Tag Clickable
情報や項目にラベルで分類し、認知不可を下げるために使用します
フィルターで使用するクリック可能なタグとして使用します

# Example
<Canvas of={tagClickableStories.Basic} />
<Controls /> 

# Usage
- クリック可能なタグは将来的に複数オブジェクトで流用が可能なデータに使用します
    - ※現在は単一オブジェクトで使用
    - 例：FO・トレンドフィルタ
