# Tag Link
情報や項目にラベルで分類し、認知不可を下げるために使用します
Tagの中にリンクが含まれ、関連する情報にたどり着くことができます

# Example
<Canvas of={tagLinkStories.Basic} />
<Controls />

# Usage
- 情報や項目にラベルで分類しリンク先を示します
    - 例：開示資料のニュースのメール通知 ダッシュボード元へのリンク
